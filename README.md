
# FindHospital  

Tìm kiếm dịch vụ y tế

Requirements
============

* PHP >= 8.1.0 & MariaDB >= 10.4.25
* or XAMPP >= m8.1.12 
* Composer >= 2.4.3


Guide
============
Migrate data

    php artisan migrate


Seeding

    php artisan db:seed UserSeeder


Run

    php artisan serve
